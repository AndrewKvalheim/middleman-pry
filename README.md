# middleman-pry

[![Build Status][build-status-badge]][build-status]

Replaces [Middleman's][middleman] built-in console with [Pry][], Ã  la
[pry-rails][].

## Installation

Add to your `Gemfile`:

```ruby
gem 'middleman-pry'
```

and run `bundle install`.

## Usage

Start the Middleman console like normal:

```shell
$ middleman console
```
